# Agent Skills Directory

This directory contains the available skills for the SquashDaddy Agent Service. Each skill is designed to handle specific user intents using the FLAP (Flow-Adhering Planning) methodology.

**Description**: helper for retrieving current schedule availability and booking details.

### [Schedule New Training](./schedule-training/SKILL.md)
**Description**: Workflow for creating new private or group training sessions.
- **Triggers**: "Book a lesson", "Schedule training for next Tuesday", "I want to train with Coach Nick"

### [Reschedule Training](./reschedule-training/SKILL.md)
**Description**: Workflow for modifying existing booking times.
- **Triggers**: "I need to move my lesson", "Reschedule my 5pm session to 6pm"
